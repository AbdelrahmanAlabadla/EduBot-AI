# Bachelor of Arts in International Relations

# Program Core Requirements

| INR 200- Introduction to International Relations                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |   | Credit Hours: 3 | Prerequisite: (Co) ENG 102/ENG 200 |   |   |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | - | --------------- | ---------------------------------- | - | - |
| This course will familiarize students with the historical developments that have affected the progress of international relations theories over the last two-hundred years. This course introduces the basic concept of interdisciplinarity in International Relations. It presents the various humanities and social science disciplines that form the basis of the study of International Relations. The course engages with the big questions of our times, including but not limited to international economic interdependency, international security, humanitarian crises, alternatives to war, and state and non-state actors. |   |                 |                                    |   |   |
